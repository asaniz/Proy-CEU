# Proy-CEU
Repositorio de los diferentes proyectos a destacar realizados a lo largo de la carrera 

## Servidor y cliete de chat 
Arquitectura cliente-servidor en C con sokets que conectan varios clientes a un chat compartido; gestión concurrente de conexiones validada en una misma máquina 
(+ modificacion para un consultor de noticias temporal donde el usuario puede publicar dichas noticias y leer las publicadas)

## Diseño y optimización de una base de datos en MySQL
Modelado entidad-relación esquema relacional y consustal SQL (JOINs, subconsultas, agregación)

## Sitio web de una tienda de ropa
Maquetado frontend en HTML y CSS (catálogo, ficha de producto y carrito), dieseño y estructura
